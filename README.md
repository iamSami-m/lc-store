# 🛍️ LC Store

A fully responsive e-commerce application built with React featuring customer and admin dashboards.

## 🌐 Live Demo

https://silly-dusk-14ec66.netlify.app/

---

## ✨ Features

### 👤 Customer Panel

* User Authentication
* Product Search
* Shopping Cart
* Wishlist
* Place Orders
* Order History
* Edit Profile
* Change Password
* Simulated Payment

### 🛠️ Admin Panel

* Dashboard
* Add Products
* Edit Products
* Delete Products
* Image Upload

---

## ⚙️ Tech Stack

* React
* React Router
* Context API
* Tailwind CSS
* JSON Server
* Git & GitHub
* Netlify

---

## 🚀 Installation

```bash
git clone https://github.com/iamSami-m/lc-store.git

cd lc-store

npm install

npm run dev
```
